Community Network
____________________________________________________________________
Communication platform featuring chat, interactive maps, and video calls.

Requirements:
    • PHP 8.0+
    • MySQL 5.7+ / MariaDB 10.3+
    • Web server (Apache/Nginx) with mod_rewrite
    • Modern browser (Chrome/Firefox) for WebRTC
________________________________________________________________________________

Installation

1. Copy Files
   	Copy the entire folder into your web server's document root (e.g., /var/www/html/community-network/).

2. Database configuration
	Edit `config.php`:
		```php
		define('DB_HOST', 'localhost');
		define('DB_NAME', 'community_network');
		define('DB_USER', 'root');     // ← user MySQL
		define('DB_PASS', '');         // ← password MySQL
		```

3. Run setup
	Open your browser and navigate to: http://localhost/community-network/setup.php
	This script will create the database, tables, and the initial admin account.

    ⚠️ IMPORTANT: Delete setup.php immediately after installation!
   
4. Default Admin Credentials
	Username: admin
	Password: Admin123!

    Note: Change your password immediately via the admin panel!

Feautures

### Chat
	- Global Chat: Visible to all online users.
	- DM (Direct Message): Click a user in the sidebar to open a private chat.
	- Mechanism: Polls for new messages every 2 seconds.

### Map
	- Powered by Leaflet.js + OpenStreetMap (No API key required).
	- Click anywhere on the map to drop a pin with an optional message.
	- Right-click on your own pin to remove it.
	- Automatic updates every 15 seconds.

### Video calls
	- WebRTC peer-to-peer (video + audio).
	- Click the 📞 icon next to a user in the sidebar to call.
	- Signaling via DB (polling every 1.5s).
	- Controls: Mute, toggle camera, end call.
	- Uses Google STUN servers (works on local networks and the internet).

### Token-based Registration
	- Registration requires an invite token generated by an admin.
	- Admins generate tokens via admin.php.
	- Each token is for single-use only.
	- Used tokens track who used them and when.

### Admin panel
	- Generate or delete invite tokens.
	- User list with "last seen" timestamps.
	- Promote or demote admin users.
	- Delete user accounts.
